# FUTURE_DS_01
Business Sales Performance Analytics

*PROJECT DESCRIPTION*

This project analyzes retail business performance by examining sales, profit, and customer behavior. It aims to identify key patterns such as top-performing products, profitable regions, and the impact of pricing strategies like discounts.

The analysis provides insights into how different parts of the business contribute to overall performance and highlights opportunities for improving decision-making and profitability.

*OBJECTIVES*

Clean and preprocess raw data
Perform exploratory data analysis (EDA)
Identify trends, patterns, and relationships
Visualize key business metrics
Build an interactive Power BI dashboard
*Data Cleaning & Preparation
Data preprocessing was performed using pandas:
Removed duplicate records
Handled missing values
Converted data types (dates, numeric fields)
Standardized categorical variables
EDA was conducted using matplotlib and seaborn to understand the structure and behavior of the data.

*SKILLS LEARNED*

Data cleaning and preprocessing
Exploratory data analysis (EDA)
Data visualization
Business intelligence and dashboard design
Analytical thinking and insight generation
*TOOLS USED*
Python (pandas, matplotlib, seaborn)
Jupyter Notebook
Microsoft Power BI

*KEY INSIGHTS*

Sales distribution is right-skewed, with most transactions in lower ranges
Profitability varies across regions and product categories
Higher discounts tend to reduce profit margins
A small number of customers contribute significantly to total revenue

*CONCLUSION*

This project demonstrates the ability to work with real-world data, extract meaningful insights, and present findings through interactive visualizations. It highlights the importance of combining technical skills with business understanding to support data-driven decision-making.
